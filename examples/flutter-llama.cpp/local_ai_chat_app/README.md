# Local AI Chat App

A Flutter-based AI chatbot application that uses the `onenm_local_llm` library to provide local LLM-based chatbot functionality. This app allows users to interact with an AI chatbot in real-time.

## Features

- Real-time AI chatbot powered by `onenm_local_llm`.
- User-friendly interface with a dark theme.
- Loading indicator while the AI processes user input.

## Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) installed on your system.
- A compatible development environment (e.g., Android Studio, VS Code).

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/SxryxnshS5/onenm_local_llm.git
   cd examples/flutter-llama.cpp/local_ai_chat_app
   ```

2. Install dependencies:

   ```bash
   flutter pub get
   ```

3. Run the app on an Android device or emulator:
   ```bash
   flutter run
   ```

### Usage

1. Launch the app on your Android device or emulator.
2. Type your message in the input field at the bottom of the screen.
3. Press the send button to interact with the AI chatbot.
4. A loading indicator will appear while the AI processes your input.
