# Local AI Summarizer

A Flutter-based application that uses the `onenm_local_llm` library to summarize long texts using a local LLM. Users can paste long texts, and the app will generate concise summaries.

## Features

- Summarize long texts using a local LLM.
- User-friendly interface with a dark theme.
- Loading indicator while the LLM processes the input.

## Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) installed on your system.
- A compatible development environment (e.g., Android Studio, VS Code).

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/SxryxnshS5/onenm_local_llm.git
   cd examples/flutter-llama.cpp/local_ai_summarizer
   ```

2. Install dependencies:

   ```bash
   flutter pub get
   ```

3. Run the app on an Android device or emulator:

   ```bash
   flutter run
   ```

### Usage

1. Launch the app on your Android device or emulator.
2. Paste the text you want to summarize into the input field.
3. Press the "Summarize" button to generate a summary.
4. A loading indicator will appear while the LLM processes the input.
